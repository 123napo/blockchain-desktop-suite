## 🧑‍💻 Example Profile Configuration

Single-file YAML lets you tailor the wallet's behavior:

    profile:
      name: "Satoshi Explorer"
      auto_fiat_conversion: true
      preferred_language: "ja"
      default_view: "assets"
      spending_thresholds:
        daily_limit_sat: 100000
        warning_threshold: 80000
      ai_assistants:
        - provider: "OpenAI"
          api_key: "your-api-key-here"
        - provider: "Claude"
          api_key: "your-claude-key"
      hardware_wallets:
        enabled: true
        type: "Trezor"
        path: "/dev/hwwallet0"
